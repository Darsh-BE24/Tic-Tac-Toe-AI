# Tic-Tac-Toe AI using Minimax and Alpha-Beta Pruning

An unbeatable console-based Tic-Tac-Toe AI built in Python using the Minimax algorithm with alpha-beta pruning for optimal move selection and decision optimization.

The AI evaluates all possible game states recursively and guarantees optimal gameplay, ensuring it never loses against a human player.

---

# Features

* Unbeatable AI opponent
* Minimax algorithm implementation
* Alpha-beta pruning optimization
* Recursive game-tree search
* Human vs AI gameplay
* Input validation and error handling
* Winner and draw detection
* Replay functionality
* Console-based interactive interface

---

# Concepts Used

* Recursion
* Backtracking
* Game-tree search
* Decision optimization
* Alpha-beta pruning
* Adversarial AI
* State-space exploration
